📜 維雷絲陸品行｜保固政策公告

🔹 基本說明

  • 本保固服務為 加購制 🛒，需在下單時或出貨前同時購買保固卡才生效。
  • 保固內容僅限於 維修或退換貨 🔧，一切以原商家／官方保固政策為準。
  • 保固服務不含人為損壞、外觀瑕疵或不可抗力因素（例：運輸損傷、自然災害）。

💳 保固卡種類與內容

  • 1️⃣ 入門保固卡：NT$199，適用商品售價 NT$1,000 以下，90 天保固
  • 2️⃣ 基本保固卡：NT$599，適用商品售價 NT$5,000 以下，180 天保固
  • 3️⃣ 進階保固卡：NT$1,999，適用商品售價 NT$20,000 以下，365 天保固

📬 申請流程

  1. 聯繫客服 → 提供訂單編號、保固卡憑證、問題描述與照片/影片
  2. 客服審核 → 確認是否符合保固條件
  3. 寄回檢測 → 檢測後依原商家／官方保固流程處理
  4. 處理時效 → 約 7～21 個工作天（依原商家進度為準）

⚠️ 注意事項

  • 保固卡需與原訂單綁定，不可轉讓或重複使用
  • 請妥善保存保固卡資訊，遺失恕不補發
  • 保固一律以原商家／官方保固規範為主，本店僅協助代辦流程
  • 加購保固卡時，建議與商品同單購買，避免遺漏
